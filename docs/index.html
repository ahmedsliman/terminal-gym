Everything in one terminal

No browser tabs, no context switching. Open a mission, read the exercises, and practice — all in the same window.

❯_

Live bash terminal

A real PTY shell embedded in the terminal panel. Switch between reading and typing with a single keypress.

≡

19 structured missions

From basic commands to awk, jq, and shell scripting — each mission has a concept brief, paginated exercises, and worked solutions.

✓

Progress tracking

Per-exercise grading based on your shell history. See exactly which commands you've practiced and what's left.

◈

4 real-world projects

Apply what you've learned — build a log analyzer, backup script, user audit tool, and dotfiles manager.

⌨

Vim-style navigation

Move through missions with j/k, switch panels with 1/2/3, and never take your hands off the keyboard.

◎

Zero dependencies

Pure Python 3 + bash. No pip installs, no Docker, no internet connection required. Clone and go.

Get started in 30 seconds

1

Clone the repo

$ git clone https://github.com/ahmedsliman/terminal-gym
2

Launch the TUI

$ cd terminal-gym && make start
3

Pick a mission and start typing

Navigate with j / k, open a mission with Enter, jump to the terminal with 3.

Requires Linux · bash · make · python3
