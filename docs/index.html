Features

Everything in one window

No context switching. Read the concept, do the exercise, run the command — all without leaving your terminal.

❯_

Live bash terminal

Real PTY shell embedded in the bottom panel. One keypress to switch between reading and typing.

✓

Auto grading

Exercises are graded from your shell history. The UI marks commands done as you run them.

≡

19 structured missions

Each mission has a concept brief, paginated exercises, worked solutions, and a practice session.

◎

Zero dependencies

Pure Python 3 + bash. No pip installs, no Docker, no internet. Clone and run immediately.

Get started

Up and running in 30 seconds

Requires Linux, bash, make, and python3. Nothing else.

bash
$ git clone https://github.com/ahmedsliman/terminal-gym
$ cd terminal-gym && make start
Requires Linux bash make python3
